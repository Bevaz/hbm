# hbm config disable
***

```markdown
Disable hbm feature

Usage:
hbm config rm [name] [flags]

Aliases:
  remove, disable, dis, off
```

## Examples

```bash
# hbm config ls
NAME
authorization
# hbm config disable authorization
# hbm config ls
```
